# Dashboard Formation

Tableau de bord professionnel pour la gestion des apprenants et des formations.

## Technologies
- PHP 8.x
- MySQL
- HTML/CSS/JavaScript
- Bootstrap 5

## Installation
1. Cloner le projet dans `C:\laragon\www\dashboard-formation`
2. Importer la base de données depuis `sql/database.sql`
3. Démarrer Laragon
4. Accéder à `http://localhost/dashboard-formation`

## Connexion par défaut
- Username: admin
- Password: admin
